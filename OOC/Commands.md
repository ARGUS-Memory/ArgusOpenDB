# Commands and Shortcuts

> **Out-of-character notice:** This document is a reference for BYOND game commands and keyboard shortcuts.

---

## Chat Commands

| Command | Function |
|---|---|
| `say "text"` | Speak in character to nearby players |
| `say ";text"` | Broadcast on your equipped headset |
| `say ":h text"` | Speak on your department radio channel |
| `whisper "text"` | Whisper; distorted by distance, inaudible beyond adjacent tiles |
| `me "text"` | Custom emote visible to nearby players |
| `subtle "text"` | Custom emote visible only to adjacent players |
| `ooc "text"` | Out-of-character broadcast to all players. Do not discuss current round IC events in OOC. |
| `adminhelp "text"` | Private message to admins. Use for reporting rule violations and bugs. |

**Formatting in say:** Use `|text|` for *italics*, `+text+` for **bold**, `_text_` for underline.

**Radio channel keys:**

| Key | Channel |
|---|---|
| `:c` | Command |
| `:n` | Research / Science |
| `:s` | Security |
| `:m` | Medical |
| `:u` | Cargo |
| `:e` | Engineering |
| `:b` | Binary (AI and Cyborgs only) |

## Action Commands

| Command | Function |
|---|---|
| `pull` | Drag an object or person behind you. Right-click > Pull, or Ctrl + Click. |
| `point` | Displays a red arrow on the targeted tile. Right-click > Point. |
| `examine` | Displays description and visible equipment of the target. Right-click > Examine. |
| `notes` | Opens your character's personal notes. Traitor objectives and uplink codes appear here. |
| `add note "text"` | Adds text to your personal notes. |
| `drop` | Drops the held item. |
| `throw` | Enters throwing mode; click a direction to throw. |

---

## System Commands

| Command | Function |
|---|---|
| `vote` | Starts or participates in a crew vote (Restart or Crew Transfer). |
| `who` | Lists all currently logged-in players by BYOND key. |
| `adminwho` | Lists currently logged-in admins. |
| `respawn` | Respawns your character if respawn is enabled. |
| `rules` | Opens the server rules window. |
| `changelog` | Opens the changelog window. |
| `toggle soundscape` | Enables or disables ambient audio. |
| `.ping` | Measures your round-trip connection time to the server. |

---

## AI / Cyborg Commands

| Command | Function | Available To |
|---|---|---|
| `list installed modules` | Lists and equips/unequips module items. First use requires module selection. | Cyborg |
| `call the shuttle` | Calls the emergency shuttle without a Communications Console. Cannot send it back. | AI |
| `lockdown` | Triggers all fire alerts simultaneously. | AI |
| `track with camera` | Follows a target with the nearest camera. Checks every 5 seconds if target leaves camera coverage. | AI |
| `cancel camera view` | Exits camera tracking or observation mode. | AI, Ghost |
| `teleport` | Teleports to map locations. | Ghost |
| `observe` | Focuses camera on a character or object. | Ghost |

## Keyboard Shortcuts

### Hotkey Mode

Press **Tab** to toggle. In hotkey mode, letter keys are bound to actions instead of chat.

| Key | Action |
|---|---|
| Tab | Toggle hotkey mode |
| W | Move up |
| A | Move left |
| S | Move down |
| D | Move right |
| Q | Drop held item |
| E | Equip |
| R | Throw |
| T | Say |
| Y | Whisper |
| 5 | Emote |
| 6 | Subtle emote |
| X | Swap active hand |
| Z | Activate held item |
| J | Toggle aiming mode |
| F | Cycle intent counterclockwise |
| G | Cycle intent clockwise |
| 1 | Help intent |
| 2 | Disarm intent |
| 3 | Grab intent |
| 4 | Harm intent |
| Ctrl + Click | Drag |
| Shift + Click | Examine |
